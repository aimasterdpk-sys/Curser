API Quickstart

Use these endpoints to manage your tournament.

POST /tournaments {"name":"My Cup","game_mode":"squad"}
POST /teams/tournament/{tournament_id} {"name":"Team A","players":[{"name":"P1"}]}
POST /matches/tournament/{tournament_id} {"map_name":"Bermuda","match_no":1}
POST /results/match/{match_id} {"team_id":1,"placement":1,"kills":8}
GET  /leaderboard/tournament/{tournament_id}
